# Changelog

User-visible changes to the hosted On-Page.ai SEO MCP server and this repo's documentation.

## 2026-07

- Documentation refresh: added `verify_installation`, MCP discovery resources, and regional targeting to the public docs.

## 2026-06

- Added `verify_installation`: a zero-credit setup check that confirms the connector is installed, authenticated, and funded. Returns server URL, credential type, current credits, route costs, and the proof fields that real scan outputs include.
- Scan tools now surface an explicit notification when scan credits are exhausted.
- The server now publishes discovery resources over MCP: customer report JSON schemas (standard and lite) and interpretation guides for on-page relevance and page-experience benchmark results.

## 2026-05

- Initial public documentation release.
