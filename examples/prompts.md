# Example Prompts

Use these prompts after connecting the On-Page.ai MCP server.

## Entity Gaps

```text
Use On-Page.ai to scan https://example.com/page for "target keyword".
Return the top missing entities, explain why each gap matters, and suggest
minimal edits to existing sentences.
```

## Competitor Coverage

```text
Run a deep On-Page.ai scan for https://example.com/page with keyword
"target keyword". Compare the recurring competitor gaps and prioritize the
fixes that would most improve relevance.
```

## Internal Links

```text
Use On-Page.ai to scan https://example.com/page for "target keyword".
Build an internal-linking plan using the recommended source pages, and suggest
natural anchor placements.
```

## Quick Bulk Review

```text
Run lite On-Page.ai scans for these URLs against their target keywords. For each
page, return only the strongest missing entities and the competitor coverage
pattern behind them.
```

## Page Experience

```text
Run a deep On-Page.ai scan for https://example.com/page with keyword
"target keyword". If a page-experience benchmark is present, compare the target
page against the top competitors and explain which speed issues matter most.
```
