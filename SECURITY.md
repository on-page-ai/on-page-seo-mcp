# Security Policy

This public repository is for install docs, examples, and MCP directory
metadata only.

Do not open issues or pull requests that include:

- API keys, OAuth tokens, or bearer tokens
- customer URLs that are not safe to share publicly
- private reports or scan outputs
- environment files
- internal deployment details
- private server source code

To report a security issue privately, email:

```text
hello@on-page.ai
```

Use the subject line:

```text
Security report: On-Page.ai MCP
```
